# AI SaaS Price Compare Extension

一个零构建的 Manifest V3 浏览器插件，用于手动录入第三方 AI SaaS 网站的套餐、积分包和模型消耗规则，并把不同平台的模型成本统一换算为 CNY、USD 等目标币种进行对比。

## 适用场景

很多 AI SaaS 网站不是直接写“每张图多少钱”或“每 1K token 多少钱”，而是采用：

```text
先购买积分包 -> 模型每次调用消耗积分 -> 再折算真实货币成本
```

本插件的核心目标就是把这条链路拆成用户容易填写的步骤：

```text
网站价格信息 -> 套餐/积分包 -> 模型消耗积分 -> 自动换算单价 -> 生成对比表
```

## 加载插件

1. 打开 Chrome 或其他 Chromium 浏览器。
2. 进入 `chrome://extensions`。
3. 打开右上角 `Developer mode`。
4. 点击 `Load unpacked`。
5. 选择当前项目根目录：`AIPriceCompareTool`。
6. 打开插件的 `Options` 页面。

## 使用流程

### 1. 录入网站

在左侧 `网站` 中添加第三方平台，例如即梦、OpenAI、Runway、Kling 等。

需要填写：

- 网站名
- 默认币种
- 备注，可选

默认币种用于帮助你记录该网站常用的原始价格币种，最终结果仍可统一换算到目标币种。

### 2. 录入套餐或积分包

在左侧 `套餐` 中添加该网站的购买方案。

如果网站是“价格购买积分”，请这样填写：

```text
套餐名：199 元 / 20000 积分
价格：199
币种：CNY
积分数量：20000
```

系统会自动计算：

```text
每 1 积分成本 = 套餐价格 / 积分数量
```

如果某个平台是订阅套餐，也可以把一个订阅周期内可用的积分数量填入 `积分数量`。

### 3. 录入模型

在左侧 `模型` 中添加要对比的模型。

需要填写：

- 模型名
- 模型类型：`text`、`image`、`video`、`audio`

模型类型会决定后续填写消耗规则时显示哪些单位，例如图片按“每张”，文本按“每 1K 输入/输出 token”。

### 4. 录入模型价格

在左侧 `模型价格` 中把网站、套餐和模型关联起来。

推荐优先使用 `积分换算` 模式：

1. 选择网站。
2. 选择模型。
3. 计费模式选择 `积分换算：先买积分，模型每次消耗积分`。
4. 选择该模型使用的积分包。
5. 填写模型每单位消耗多少积分。

示例：

```text
网站：Demo SaaS
套餐：39 CNY / 400 积分
模型：gpt-image-1
类型：image
每张图片消耗：20 积分
```

系统会自动预览：

```text
每 1 积分 = 0.0975 CNY
每张图片 = 1.95 CNY
```

如果网站直接给出单价，则使用 `直接价格` 模式，直接填写每单位价格和币种。

### 5. 设置汇率和目标币种

在中间栏 `整体参数` 中设置汇率和目标结果币种。

默认内置：

```text
CNY = 1
USD = 7.2
HKD = 0.92
```

如果希望所有结果都换算为 USD，把 `默认结果币种` 改成 `USD` 即可。

### 6. 生成对比表

在中间栏 `对比设置` 中选择要对比的平台和模型，然后点击 `生成对比表`。

结果表会展示：

- 网站
- 对比类型
- 模型
- 计费方式
- 套餐
- 消耗积分/单价
- 原币种单价
- 目标币种单价
- 本次成本

对比类型只显示 `文本`、`图片`、`视频`、`音频`，不会再展开显示不相关的输入、输出、图片、视频、音频明细。

## 快速验证示例

1. 打开 `Options Page`。
2. 点击 `载入示例数据`。
3. 点击 `生成对比表`。
4. 确认结果表中出现 `gpt-image-1`。
5. 确认目标币种单价和本次成本均为 `1.95 CNY`。

示例数据对应的计算过程：

```text
39 CNY / 400 积分 = 0.0975 CNY / 积分
0.0975 CNY * 20 积分 / 张 = 1.95 CNY / 张
```

## 数据导入导出

插件数据保存在浏览器本地。

- 点击 `导出 JSON` 可以备份当前录入的网站、套餐、模型价格和参数。
- 点击 `导入 JSON` 可以恢复之前导出的数据。
